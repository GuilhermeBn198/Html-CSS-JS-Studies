Como Funciona Html e CSS

primeiramente não são linguagens de programação

HTML

HTML EH UMA LINGUAGEM FOCADA EM CONTEÚDO, podendo ser textos, imagens, listas.

Existem várias tags que podemos usar no html como as tags h para títulos(sendo dividido em 6 variantes de 1 a 6), as tags < p> < /p> para parágrafos e as tags < img src="" alt="" > que não contém fechamento de tag igual as anteriores

exemplos de utilização de html entities


um código HTML eh composto por 2 seções distintas, o HEAD que fica as configurações do html e o BODY que fica o conteúdo

CSS

O CSS TEM FOCO EM DESIGN, consegue alterar todo o conteúdo de uma página web, com sombras, tamanhos, formatos e variantes

para mexer com CSS eh preciso abrir uma tag style com seletores por ex

h1{
font-family:Arial;
font-size:20pt;
color:blue;
}
h1 sendo o seletor... os comandos dentro do seletor são propriedades, que são definidas por valores após 2 pontos, COLOR eh uma propriedades que também tem um valor a definir.. no caso BLUE
